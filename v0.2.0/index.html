FresnelIntegrals

A quick Julia library for calculating Fresnel Integrals using the error function from SpecialFunctions.jl.

Functions

FresnelIntegrals.fresnel — Method
fresnel(z::Number)

Calculate the normalized cosine and sine fresnel integrals.

See also fresnels, fresnelc.

source
FresnelIntegrals.fresnelc — Method
fresnelc(z::Number)

Calculate the normalized Fresnel cosine integral

\[C(z) = \int_{0}^{z} \cos{\left(\frac{\pi t^2}{2}\right)} \, \mathrm{d}t\]

for the number $z$.

source
FresnelIntegrals.fresnels — Method
fresnels(z::Number)

Calculate the normalized Fresnel sine integral

\[S(z) = \int_{0}^{z} \sin{\left(\frac{\pi t^2}{2}\right)} \, \mathrm{d}t\]

for the number $z$.
